Why I built this website

I created this website primarily for personal use, as a database and automation tool for me and some friends. We use it mostly to analyze and optimize things remotely, which are some of my interests. However, I recently updated it to be more user-friendly and included my CV and previous work experience for networking purposes. It's important to note that I built everything from scratch without any formal programming or web development education (My majors are Economics and International business). So if you notice any errors, please don't hate too hard, is a work in progress.
